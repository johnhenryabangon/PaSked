# PaSked - Quick Start Guide

## What You Just Downloaded
This is the complete PaSked Sports Court Booking System - ready to run!

## Quick Setup (5 minutes)
1. Extract this ZIP file
2. Move the "PaSked" folder to C:\xampp\htdocs\
3. Start XAMPP (Apache + MySQL on port 3307)
4. Create database "pasked" in phpMyAdmin
5. Import the database_setup.sql file
6. Visit http://localhost/PaSked/
