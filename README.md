# 🏐🏀 PaSked – Sports Court Reservation System

**PaSked** is a web-based reservation platform designed to simplify the booking and management of sports courts in Metro Manila. It focuses on basketball and volleyball courts commonly found in local community venues, while also supporting non-sports events such as tournaments, trainings, and social gatherings.

The system provides a **fast and user-friendly booking experience** by allowing users to reserve courts without requiring account registration. Instead, essential booking details and ID verification are used to ensure legitimacy and prevent spam reservations.

On the administrative side, PaSked offers a dedicated dashboard for court facilitators, enabling them to efficiently manage bookings, monitor schedules, and avoid conflicts through a visual weekly schedule overview.

---

## ✨ Key Features

- 🏀 **Court Browsing System**  
  View available courts with photos, rates, location, and supported activities  

- 📅 **Direct Booking (No Login Required)**  
  Simple reservation process using contact details and schedule input  

- ⚠️ **Conflict Detection System**  
  Prevents overlapping bookings with real-time validation  

- 📊 **Weekly Schedule Overview**  
  Visual chart showing availability (Available, Booked, Declined)  

- 🛠️ **Admin Dashboard**  
  Manage bookings, approve/decline requests, and review uploaded IDs  

- 🪪 **ID Verification Upload**  
  Adds a layer of security and reduces fake or spam bookings  

- 📱 **Responsive Design**  
  Works smoothly on both desktop and mobile devices  

---

## 🧠 Purpose of the Project

PaSked was developed as an academic project to address real-world scheduling issues experienced by athletes and court facilitators. The system promotes **digitalization of sports facilities**, improves accessibility, and enhances organization in local community courts.

---

## ⚙️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Local Development:** Laragon  
- **Deployment:** AwardSpace (Free Hosting)

---

## ⚠️ Limitations

- No user account system (intentionally designed for simplicity)  
- No integrated online payment  
- Email notifications (PHPMailer) are not functional due to free hosting SMTP restrictions
